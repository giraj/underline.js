Underline.js

Underline.js is a more flexible underline for the web. It allows you to ignore characters, specify margins, stroke width and colors. More options could probably be extended.

Currently, the biggest issue is underlining text spanning more than one line, which doesn't work at the moment.

However, it works for single-line titles and other single-line purposes. An option I would like to add is to be able to specify "padding" around certain ignored characters; the 'q' can be quite spacious from the left-hand side, for example.

The docs are the README on github, so head over there.
