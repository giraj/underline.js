Underline.js

Underline.js is used as a more flexible, customizable underline. With it, you can specify characters not to underline, as seen in the heading, and other features such as line-width, margin, color.

This doesn't work really if the text spans several lines between "break-characters" (g, y, etc.) due to how this is done. Could be fixed, but I am not working on it.

However, for single-lined titles it works. It would be nice to be able to specify "spacing" or "padding" around certain letters, f.ex. the 'q' can be quite spacious from the left-side when using this.

The docs are the README on github, so head over there.
